# 基于Python的图书馆管理系统 python391

## 项目概述

本项目是基于Python3.7与Django框架，搭配MySQL5.7和Vue.js构建的图书馆管理系统。系统旨在为管理员和用户提供一个便捷、高效的管理和使用平台。

## 技术栈

- **后端**: Python3.7, Django
- **数据库**: MySQL5.7
- **前端**: Vue.js

## 功能模块

### 系统角色

- 管理员
- 用户

#### 管理员功能

1. 首页查看
2. 个人中心管理
3. 用户管理
4. 图书馆管理
5. 图书信息管理
6. 图书类型管理
7. 借阅信息管理
8. 归还信息管理
9. 续借信息管理
10. 罚金信息管理
11. 留言板管理
12. 系统管理

#### 用户功能

1. 首页查看
2. 个人中心管理
3. 借阅信息管理
4. 归还信息管理
5. 续借信息管理
6. 罚金信息管理
7. 留言板互动
8. 我的收藏管理

## 运行环境

- 后端：Python3.7及以上版本环境，Django框架安装
- 数据库：MySQL5.7及以上版本
- 前端：现代浏览器支持

## 部署步骤

1. 环境准备：安装Python3.7，MySQL5.7，及相应的Python库和工具
2. 源码部署：将后端代码放置于服务器指定目录，并配置数据库连接
3. 前端部署：将Vue.js构建的静态文件放置于Web服务器目录
4. 系统启动：运行Django项目，使服务生效
